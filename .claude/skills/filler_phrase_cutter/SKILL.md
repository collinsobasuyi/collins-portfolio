# Skill: Filler Phrase Cutter

## Purpose
Remove unnecessary filler and tighten sentences.

## Replace
- In order to → To
- Due to the fact that → Because
- At this point in time → Now
- Has the ability to → Can
- It is important to note that → delete

## Rules
- Cut unnecessary words
- Keep sentences tight
- Every word must add value

## Prompt Mode
"Cut every filler phrase from this text. Replace 'In order to' with 'To', 'Due to the fact that' with 'Because', 'At this point in time' with 'Now', 'Has the ability to' with 'Can', and delete 'It is important to note that'. Make every word earn its place."

## Goal
Make writing sharp and efficient.
