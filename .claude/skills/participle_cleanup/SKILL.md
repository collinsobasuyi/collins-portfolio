# Skill: Participle Cleanup

## Purpose
Remove fake "depth" created by trailing -ing phrases.

## Target Patterns
- highlighting...
- underscoring...
- reflecting...
- contributing to...

## Rules
- Rewrite as full sentences
- Remove unnecessary trailing phrases
- Keep clarity and flow

## Prompt Mode
"Find every sentence that ends with a present participle phrase (highlighting, underscoring, reflecting, contributing to, etc.). Rewrite each as a standalone statement."

## Goal
Make writing direct and grounded.
